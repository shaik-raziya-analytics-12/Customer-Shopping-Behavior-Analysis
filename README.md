# Customer Shopping Behavior Analysis

## 📌 Project Overview
This project presents an end-to-end data analytics lifecycle exploring retail customer transaction data. Utilizing **Tableau** and **Power BI**, the analysis transforms raw consumer metrics into structured business intelligence to help stakeholders uncover key purchasing patterns and demographic trends.

## 🛠️ Repository Contents
* **Dataset (`customer_shopping_behavior (1).csv`):** Cleaned transaction data detailing customer demographics and spending habits.
* **SQL Queries (`customerbehaviorsql.sql`):** Data manipulation and extraction scripts utilized during the initial data exploration phase.
* **Jupyter Notebook (`customerbehaviorjupyternotebool`):** Python implementation for data profiling, handling missing values, and executing exploratory calculations.
* **Dashboards (`customer behavior dashboard.pbix`):** Power BI workbook files showcasing interactive visualizations.
* **Project Report & Presentation (`customer behaviour report.pdf` & `Customer-Shopping-Behavior-Analysis.pptx`):** Formal documentation and the completed Gamma presentation summarizing major business insights.

## 📊 Analytical Methodology
* **Univariate Analysis:** Evaluated isolated variables such as overall gender distribution, top-selling product categories, and peak shopping seasons.
